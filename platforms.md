# Platforms

ROOST and the community interact across multiple platforms. These are the officially-recognized platforms and spaces that we currently use.

## <i class="fa-brands fa-github"></i> GitHub

ROOST open source projects are hosted on the [@roostorg](https://github.com/roostorg) organization on GitHub. Each GitHub repository is where you'll find the project's source code, issue tracking and project management, pull requests and code review, and related discussions. If the project contains documentation, it can be found directly in the code repo and hosted on the web at the repo's listed website.

[ROOST-wide discussions](https://github.com/orgs/roostorg/discussions) can also be found on the ROOST org itself.

### Mirrors

We mirror read-only copies of our repositories on the following code forges:

- [@roostorg](https://gitlab.com/roostorg) on GitLab
- [@roost.tools](https://tangled.org/roost.tools) on Tangled

These mirrors do not contain issues or pull/merge requests, but do have a copy of all git branches and their commits.

## <i class="fa-brands fa-discord"></i> Discord

We operate the [ROOST Discord server](https://discord.gg/5Csqnw2FSQ) for real-time communication and collaboration with ROOST staff and the community. When joining, you can select your specific areas of interest to tailor your experience. You can also directly join these project-specific channels for collaboration with contributors and adopters:

- [#coop](https://discord.gg/97GQqtNAkE)
- [#osprey](https://discord.gg/AwxZ9K6MsU)
- [#model-community](https://discord.gg/UXmBqy7kFX)

Be sure to introduce yourself in `#welcome` and ask questions or chat about anything related to ROOST or online safety in `#general`.
